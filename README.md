# H6-server

## install
* git clone
    * $git https://github.com/kingdom0608/HANDAM.git
* npm install    
    * $npm install -g typescript
    * $npm i @types/node
    * $npm install
    * $tsc --p tsconfig.json
* redis install
    * linux
        * $sudo apt-get install redis-server
    * unix
        * $brew install redis
        
## start    
* redis start
    * $redis-server                
* server start
    * $npm start
